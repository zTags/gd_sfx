# GD SFX

(we should find a better name for this project)

GD SFX is a Geometry Dash SFX Library explorer.

## Features

- [x] SFX Playback
- [x] Categories
- [x] Download and delete
- [x] Favourites
- [x] Searching
- [x] SFX Credits
- [ ] Statistics
- [ ] Mass download and mass delete
- [ ] Sort by name / duration / id / size
- [ ] Change speed and pitch
- [ ] Change volume
